Flask + MNIST

Draw a digit and let the saved model read and predict it live.

This interface converts your drawing into a centered 28 x 28 image, then automatically checks whether mnist.pkl expects raw pixel values or PCA-transformed features. Based on that, it applies the required preprocessing and sends the data to the model to generate the prediction.

Prediction Flow

Canvas to Flask

JavaScript captures the user's drawing from the canvas, Flask processes and formats the pixel data, and the trained ML model predicts and returns the digit.

Loaded Model

The application uses a K-Nearest Neighbors (KNN) classifier to make predictions.

How it works:

When you draw a digit, the model compares it with similar examples it has seen before and predicts the digit based on the closest matches.

Pipeline: The processed input is passed directly into the KNN model to generate the final prediction.

Feature Mapping

Your drawing is converted into 784 pixel values (28 x 28 image), just like the original training data. These pixel values are then passed through the same preprocessing steps used during training: StandardScaler -> PCA. PCA reduces the 784 features down to {{ meta.expected_model_features if meta and meta.expected_model_features else 300 }} important components, which the model uses for prediction.
